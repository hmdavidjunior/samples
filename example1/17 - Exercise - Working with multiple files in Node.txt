Part 1 - Your own module

    Create a script named app.js
    Create another script named calculator.js
    Calculator should export a Javascript object that has 4 functions: add, subtract, multiply, divide - each function taking two parameters of type Number
    app.js should import your calculator module
    In app.js call each function of the imported module's object and print the results of the operation to the console
